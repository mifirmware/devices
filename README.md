# Devices

This repository dependency for [maker.](https://github.com/mifirmware/maker)

## Available Devices

* Mi 5 (gemini)
* Mi 5s (capricorn)
* Mi 5s Plus (natrium)
* Mi 6 (sagit)
* Mi MIX (lithium)
* Mi MIX 2 (chiron)

## Device JSON Scheme

Fill blanks for your device.

```
{
    "codename": "",
    "id": "",
    "name": "",
    "content_id": {
        "global": "",
        "china": ""
    }
}
```
